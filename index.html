Athena CP

A Control Panel for rAthena

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Athena Control Panel webpage.

This is a forked project from the original (non-renewal) Flux Control Panel. The purpose for this repository is to create/update a control panel that has renewal elements built in. The content will mainly be guided by the progress of rAthena, an open-source renewal emulator for Ragnarok Online.

Main features to be added: Account Clustering, Facebook and OpenID authentication, Live Vending
